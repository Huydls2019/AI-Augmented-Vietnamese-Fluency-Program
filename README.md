# AI-Augmented Vietnamese Fluency Program 🚀

An open-source LLM-driven educational framework designed to accelerate Vietnamese language acquisition through personalized conversational agent routing and automated pronunciation analysis.

## 📊 Project Impact & Metrics
* **Scale:** Successfully coached **50+ language learners** across both professional and academic tracks.
* **Timeline:** Structured within a **4-week high-intensity** fluency window.
* **Core Technology:** OpenAI/Anthropic API Orchestration, Prompt Chaining, Custom Phonetical Logs.
